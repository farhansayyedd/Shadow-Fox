# 🧑‍💻 Farhan Sayyedd — Personal Portfolio

> A premium, dark-themed personal portfolio website built with pure HTML, CSS, and vanilla JavaScript.

**🔗 Live Demo:** [farhansayyedd.github.io/Shadow-Fox/portfolio](https://farhansayyedd.github.io/Shadow-Fox/portfolio)

---

## 📸 Preview

A sleek single-page portfolio showcasing skills, projects, and contact details with a modern dark-mode aesthetic.

---

## 🚀 Features

- ⚡ **Zero Dependencies** — Pure HTML5, CSS3, and Vanilla JS, no frameworks
- 🌑 **Dark Mode Design** — Professional dark theme with vibrant accent colors
- 📱 **Fully Responsive** — Mobile-first layout, adapts to all screen sizes
- 🎨 **Smooth Animations** — CSS keyframe animations and scroll-triggered transitions
- 📄 **Multi-page Structure** — Separate pages for About, Skills, Projects, Resume, Contact
- 🔗 **Social Links** — GitHub, LinkedIn, and other profile links integrated

---

## 📁 File Structure

```
portfolio/
├── index.html          # Landing page / hero section
├── about.html          # About me page
├── skills.html         # Technical skills showcase
├── projects.html       # Project gallery
├── resume.html         # Resume / CV viewer
├── contact.html        # Contact form & links
└── farhan_resume.html  # Downloadable resume page
```

| Tech | Usage |

## 🏃 Running Locally

No build step required — just open any `.html` file directly in your browser:

```bash
# Clone the repo
git clone https://github.com/farhansayyedd/Shadow-Fox.git

# Open portfolio
cd Shadow-Fox/portfolio
# Open index.html in your browser
```
